## 🚀 About This Application

This repository contains the source code for the custom digital billing and invoice generation web application built exclusively for **Shree Adyashakti Caterers**. The tool streamlines the checkout process, automatically calculates service totals, and generates print-ready professional invoices for clients. By integrating **Firebase**, the system securely handles real-time data syncing and cloud storage for past invoices.

**🌐 Live Demo:** [https://adhyashakti-bill.netlify.app/](https://adhyashakti-bill.netlify.app/)

---

## ✨ Key Features

* **Cloud Data Management:** Firebase integration allows for saving, retrieving, and managing a history of generated bills securely in the cloud.
* **Dynamic Item Entry:** Easily add catering services, food items, and quantities directly to the digital invoice interface.
* **Automated Calculations:** JavaScript-powered logic automatically computes subtotals, taxes, final totals, and remaining balances without manual math.
* **Print-Ready Formatting:** The UI is specifically optimized with CSS media queries to instantly convert the webpage into a clean, professional A4 PDF or physical printout.
* **Zero-Setup Operation:** Fully web-based. No software installation required for the proprietors (Kamleshbhai Patel & Maulin Patel) to generate bills on the go from any device.
