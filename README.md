## 🚀 About This Application

This repository contains the source code for the custom digital billing and invoice generation web application built exclusively for **Shree Adyashakti Caterers**. The tool streamlines the checkout process, automatically calculates service totals, and generates print-ready professional invoices for clients.

**🌐 Live Demo:** [https://adhyashakti-bill.netlify.app/](https://adhyashakti-bill.netlify.app/)

---

## ✨ Key Features

* **Dynamic Item Entry:** Easily add catering services, food items, and quantities directly to the digital invoice.
* **Automated Calculations:** JavaScript-powered logic automatically computes subtotals, final totals, and remaining balances without manual math.
* **Print-Ready Formatting:** The UI is specifically optimized with CSS media queries to instantly convert the webpage into a clean, professional A4 PDF or physical printout.
* **Zero-Setup Operation:** Fully web-based. No software installation required for the proprietors (Kamleshbhai Patel & Maulin Patel) to generate bills on the go from any device.

## 💻 Tech Stack

* **Frontend UI:** HTML5 & Custom CSS (Focus on print-media styling)
* **Logic & Calculations:** Vanilla JavaScript (DOM manipulation and math operations)
